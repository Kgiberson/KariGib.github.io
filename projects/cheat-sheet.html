Ruby Enumerable Cheat Sheet

Basic Methods Description Examples
to_a Returns array of all elements (1..7).to_a
#=> [1,2,3,4,5,6,7]
{ 'a' => 1, 'b'=>2, 'c'=>3 }.to_a
#=> [["a", 1], ["b", 2], ["c", 3]]
count
count(item)
count{ |obj| block }		 Returns the number of items in the enumberable. If a block is given it counts the number of elements yielding true. arr = [1,2,5,4]
arr.count
#=> 4
arr.count(2)
#=> 2
arr.count{ |x| x % 2 == 0}
#=> 2
include? Returns true if any member of enum equals obj. Equalitty is tested using ==. Checks if a collection "includes" the item passed in as an argument. array = [1,2,3,4] array.inclue?(3)
#=> true

Iteration Methods Description Examples
each{ |x| ... }
each_entry{ |x| ... } 		Calls block once for each element in 'self', passing the element as a parameter
each_with_index(args){ |obj, i| ... } Calls block with two arguments, the item and its index, for each item in 'enum'. Given arguments are passed through to each(). hash = Hash.new
['horse', 'cat', 'dog'].each_with_index { |item, index|
hash[item] = index
}
hash #=> {'horse'=> 0, 'cat'=> 1, 'dog'=>2}
each_slice Breaks collection into a group of n sized elements. Useful to transform an array into an array of sub arrays for modifications. (1..8).each_slice(4) { |x| print x}
#=> [[1,2,3,4],[5,6,7,8]]
map Returns a new array after performing a block operation on each of the elements in the collection. It transforms the collection! (1..4).map { |x| x + 2 }
#=> [3,4,5,6]
grep(pattern)
grep(pattern) { | obj | ... }		 Returns an array of every element in enum for which pattern === element. If the block is supplied, each matching element is passed to it, and the block's result is stored in the output array. (1..100).grep(20..28)
#=> [20,21,22,23,24,25,26,27,28]
inject Easily allows users to add, multiply, and subtract through each element of a collection. (1..5).inject { | sum, element | sum + element }
#=> 15
